# Trader Hiro - Landing Page

Landing page profissional convertida do Elementor para HTML/CSS/JS puro.

## Estrutura de Arquivos

```
public_html/
├── index.html              # Página principal
├── assets/
│   ├── css/
│   │   └── styles.css      # Estilos CSS (responsivo)
│   ├── js/
│   │   └── main.js         # JavaScript com interatividade
│   └── images/
│       ├── bg-desktop.webp # Imagem de fundo desktop
│       └── bg-mobile.webp  # Imagem de fundo mobile
└── README.md               # Este arquivo
```

## Deploy

Para fazer deploy, basta fazer upload de toda a pasta `public_html` para o servidor via FTP/cPanel.

### Passos:
1. Acesse seu painel de hospedagem
2. Vá até o gerenciador de arquivos
3. Navegue até a pasta `public_html` do servidor
4. Faça upload de todos os arquivos desta pasta
5. Acesse seu domínio para visualizar
